# Primeiro Site

Atividade sugerida em aula com o objetivo de praticar componentes básicos de HTML, como títulos, imagens, âncoras e ícones, a partir de um tema pré-sorteado. No caso deste projeto, o tema foi Alcachofra, então o site apresenta uma visão básica sobre o assunto.
